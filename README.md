# RoyceLark Portfolio (static)

This is a simple static portfolio site generated from the resume provided during the conversation.
It includes an HTML page, CSS, and a copy of the uploaded PDF resume.

## How to use

1. Unzip the archive.
2. Review `index.html` and `styles.css` and edit content as needed.
3. To deploy on GitHub Pages:
   - Create a repository named `RoyceLark.github.io` under your GitHub account.
   - Commit and push the site files (all files in the unzipped folder) to the `main` branch.
   - GitHub Pages will publish the site at `https://RoyceLark.github.io/` within a few minutes.

## Alternative: Deploy from any repo using GitHub Pages
- Create a repo (e.g., `portfolio`) and push files.
- In the repo settings -> Pages, set the source to `main` branch and `/ (root)` and save.
